Mobile Developer

Mobile Developers use their programming knowledge to build mobile applications, as well as to build mobile versions of websites. Given the prevalence of mobile devices, employment opportunities for Mobile Developers are promising. Virtually every company in any industry you could think of has a mobile application so that Mobile Developers can find employment in any industry.

Skills Needed

  • Writing native mobile apps for iOS and Android
  • UI, working closely with industrial design to achieve great UX
  • Expert-level in Obj-C / Swift and Java
  • Experience in RESTful API delivering JSON or server-side development (C#)
  • Strong knowledge of Objective-C with a strong understanding of iOS architecture, frameworks and the Xcode IDE
  • Familiarity with Agile and Scrum methodologies within a highly collaborative atmosphere
